# Go Practical Exercises

This repository contains small practical exercises for learning the Go programming language.

## Overview

Each lab is a self-contained folder with simple examples to help you practice core Go concepts.

## Repository structure

- `lab_01/` — basic starters (includes `main.go` and `variables.go`).

## Prerequisites

- Go 1.18+ installed. Verify with:

```
go version
```

## Run the examples

From the repository root you can run the example in `lab_01`:

```
go run ./lab_01
```

Or run a single file (for quick testing):

```
go run ./lab_01/variables.go
```

## What you'll learn in `lab_01`

- Declaring and using variables
- Writing a simple `main` function
